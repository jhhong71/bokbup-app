# 오늘의 복붙 — Google Play 출시 가이드

비용: **$25 (1회, 개발자 계정 등록비)** — 그 외 전부 무료.
기간: 비공개 테스트 의무(테스터 12명 × 14일) 때문에 **최소 2~3주**.

## 0. 알아둘 것 (구글 정책)

2023년 11월 13일 이후 만든 **개인** 개발자 계정은 정식 공개(프로덕션) 전에
**12명 이상의 테스터가 14일 연속 참여하는 비공개 테스트**를 통과해야 합니다.
가족·친구 12명의 Gmail 주소를 미리 모아두세요.

## 1. Google Play 개발자 계정 만들기 ($25)

1. https://play.google.com/console 접속 → 구글 계정으로 로그인
2. "개인" 계정 유형 선택 → 본인 인증(신분증) → $25 결제
3. 승인까지 며칠 걸릴 수 있음

## 2. PWABuilder로 안드로이드 패키지 만들기 (무료, 10분)

1. https://www.pwabuilder.com 접속
2. URL 입력: `https://port-0-bokbup-app-mrem6lseafd21208.sel3.cloudtype.app`
3. **Package for Stores** → **Android** → **Google Play** 선택
4. 설정:
   - Package ID: `com.jhhong71.bokbup`
   - App name / Short name: 오늘의 복붙
   - Signing key: **Create new** (PWABuilder가 서명 키를 만들어줌)
5. **Download** → ZIP 안에 이런 파일이 들어 있음:
   - `*.aab` — Play Console에 올릴 앱 패키지
   - `assetlinks.json` — 도메인 소유 증명 파일
   - `signing.keystore` + 비밀번호 파일 — **절대 잃어버리면 안 됨!** (업데이트할 때 필요. 클라우드에 백업 권장)

## 3. assetlinks.json 배포 (Claude에게 맡기기)

ZIP 안의 `assetlinks.json` 내용(특히 `sha256_cert_fingerprints` 값)을 Claude에게 붙여넣어 주세요.
`public/.well-known/assetlinks.json`에 반영해서 push하면 자동 배포됩니다.
이게 있어야 앱 실행 시 상단에 주소창이 안 보여요.

## 4. Play Console에서 앱 등록

1. Play Console → **앱 만들기** (이름: 오늘의 복붙, 언어: 한국어, 무료)
2. **테스트 → 비공개 테스트** 트랙 생성 → AAB 업로드
3. 테스터 이메일 목록 추가 (12명 이상) → 테스트 링크를 테스터들에게 공유
4. **스토어 등록정보** 작성:
   - 앱 아이콘: 512×512 (`public/icons/icon-512.png` 그대로 사용 가능)
   - 그래픽 이미지: 1024×500 (Claude에게 만들어달라고 하면 됨)
   - 휴대폰 스크린샷 2장 이상 (폰에서 앱 캡처)
   - 간단한 설명 / 자세한 설명 (아래 초안 참고)
   - 개인정보처리방침 URL: `https://port-0-bokbup-app-mrem6lseafd21208.sel3.cloudtype.app/privacy.html`
5. **콘텐츠 등급 설문** (전체이용가로 나옴), **데이터 보안 섹션**: "데이터 수집 안 함" 선택
   (기기 밖으로 나가는 개인 데이터가 없음 — 익명 카운트만 집계)

## 5. 14일 후 프로덕션 신청

테스터 12명이 14일간 옵트인 상태를 유지하면 대시보드에 **프로덕션 액세스 신청**이 열립니다.
설문(테스트에서 배운 점 등) 작성 → 승인되면 **프로덕션 출시** → 심사(보통 며칠) → 스토어 공개! 🎉
